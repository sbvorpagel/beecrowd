# Beecrowd codes and docker custom judge

## How to use?

### create a new code
`make init CODE="<CODE_NUMBER>"`

### build image
`make build`

### run codes
`make run CODE="<CODE_NUMBER>"`

#### Example:
`make run CODE="1000"` to run the 1000 code of Beecrowd
